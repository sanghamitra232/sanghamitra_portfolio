# Sanghamitra Deb - Portfolio Website

Cinematic interactive portfolio built with Next.js 15, Three.js, Framer Motion.

## Quick Start

```bash
npm install
npm run dev
```
Open http://localhost:3000

---

## Content Editing - NO CODE REQUIRED

All content lives in /data/*.json files. Replace media in /public/ folders.

### Personal Info
Edit: data/personal.json

### Add a Project
Edit: data/projects.json
Drop image in: public/projects/

### Add a Certificate
Edit: data/certifications.json
Drop image in: public/certificates/

### Add Internship
Edit: data/internships.json
Drop logo in: public/internships/

### Add Industrial Visit
Edit: data/industrial.json
Drop image in: public/industrial-exposure/

### Skills
Edit: data/skills.json

### Stat Counters
Edit: data/achievements.json

## Audio
Drop MP3s in public/audios/
Reference in JSON as: "/audios/filename.mp3"

## Resume
Place at: public/resume/Sanghamitra_Deb_Resume.pdf
